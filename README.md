# meirgOS

meirgOS is a small experimental operating system written in **Rust**.  
The project is currently in a very early stage and mainly serves as a learning and experimentation platform for operating system development.

## About

The goal of **meirgOS** is to explore low-level system programming concepts such as:

- memory management
- hardware interaction
- boot processes
- kernel development
- safe systems programming with Rust

At the moment the project is in its **initial development phase**, so many features are still missing or under construction.

## Features (Planned)

- Basic kernel written in Rust
- Bootloader support
- Memory management
- Interrupt handling
- Simple drivers
- Basic shell / command interface

## Project Status

🚧 **Early development**  

This project is currently experimental and not intended for production use.
